# Ontology Visualizer (DOT Writer)

## Summary

ONT-DOT is a tool for visualizing OWL and RDF ontologies, using [Graphviz](http://graphviz.gitlab.io/).

##### Useful links:
- 
- https://graphviz.org/
- https://dreampuf.github.io/GraphvizOnline/

##### Requirements:

- Git
- Java **11**
- Maven **3+**
